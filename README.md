#### What we want to assess:

### Development

The script will run webpack with a local server on port 8080 and the mock-api on port 1337

```sh
   npm run dev
```

### Production

```sh
   npm run build
```

### Lint

If you want to make some changes and commit, please run the lint command first.

```sh
   npm run lint
```
